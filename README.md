# Auto-Docs
Script en Python Automatizador de Documentos que genera documentos Word (.docx) con portada, encabezado institucional e índice de secciones, a partir de una plantilla reutilizable.

El documento brinda una estructura genérica, tal como: 
- Indice
- Introducción
- Descripción General
- Propuesta de Planificación 
- Conclusión.

# Requisitos
pip install python-docx

# Uso
1. Coloca tu imagen de encabezado como "encabezado.jpg" dentro del directorio "AutoWord"
2. Edita los datos en el script (estudiantes, profesor, asignatura, sección, páginas siguientes)
3. Ejecuta el script.
4. El documento "Documento.docx" se ha generado en el directorio base "AutoWord".

# Tecnologías
- Python 3
- python-docx
